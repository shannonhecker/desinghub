# Design Reference Images

## 1. Zyricon AI Chat (Dark Theme — Primary Reference)
- **Layout:** Left sidebar (260px) + full-width main content
- **Sidebar:** Dark (#0c0c18), logo, "New Chat" button, Features nav (Chat/Archived/Library), Workspaces list, "Upgrade to premium" card at bottom
- **Top Bar:** Model selector dropdown, "Configuration" + "Export" buttons
- **Hero:** Glowing purple 3D orb, "Ready to Create Something New?" headline
- **Actions:** Pill-shaped chips — "Create Image", "Brainstorm", "Make a plan"
- **Input:** Large textarea with "Ask Anything..." placeholder, bottom toolbar (Attach/Settings/Options), mic + send buttons
- **Feature Cards:** 3 dark cards with icon, tag badge, title, description
- **Colors:** Deep purple (#7c3aed), violet (#a855f7), pink (#ec4899), dark bg (#0a0a14)
- **Effects:** Ambient purple glow, glass morphism, gradient borders

## 2. Miracle AI (Mobile / Light Theme)
- **Layout:** Mobile-first, centered content
- **Colors:** Soft lavender (#ede9fe), lilac gradients, pastel purple accents
- **Orb:** Centered with animated dot particles around it
- **Chat:** Rounded bubbles with purple gradient, floating action buttons
- **Actions:** "Rewrite", "Create an Image", "Make a Plan", "Analyse the Data"
- **Typography:** Clean sans-serif, large friendly text

## 3. Karla AI (Landing Page)
- **Layout:** Full-width landing page with header nav
- **Header:** Logo, nav links (Home/Features/Why/Pricing/How it Works), Contact CTA
- **Hero:** "Automate, Engage, and Grow with AI Chat" + "Try It Free" / "Watch Demo" CTAs
- **Sections:** Features grid, partner logos, pricing cards (Pro $19.99/Enterprise Custom), FAQ accordion
- **Colors:** Indigo/blue (#4f46e5) accent, clean white backgrounds
- **Cards:** Subtle shadows, rounded corners (16px), clean typography
